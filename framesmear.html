Parameter reference

memory decay
Reverb time. How long the feedback tail persists before dissolving.
new frame opacity
How aggressively each fresh frame inscribes over the buffer. Lower = longer smear.
flow X / Y
Early reflection direction. The feedback buffer drifts in this direction each frame.
feedback zoom
Room size. Above 1 the buffer expands outward; below 1 it contracts to center.
rotation
The room spins. Feedback accumulates in a spiral; strong values create vortex trails.
oscillation depth / rate
Tape wow. Flow direction breathes sinusoidally — unstable, warping accumulation.
strobe depth / rate
Gated reverb. Decay pulses between near-frozen and fast-fade, creating discrete echo stamps.
hue drift
Shimmer. The feedback buffer cycles through the spectrum while fresh content stays natural.
chroma split
Chromatic aberration on the output. R and G channels offset in opposing directions.
edge inscription
High-pass gate. Raises priority of sharp edges in the inscription mask.
light persistence
Brightness gate. Bright regions are inscribed more aggressively into the buffer.
softening
Diffusion. Blurs the buffer before re-inscribing — spreads trails, reduces definition.
