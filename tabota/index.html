A workspace for the Tabota notation language: write or paste a score and see it plotted, then compile it to microtonal MIDI or read MIDI back into Tabota. Pitch may be written as raw frequency, symbolic notes, or tuned through Scala .scl files; playback retunes ordinary MIDI notes through pitch bend so the result can travel into a normal DAW. For drawing directly instead of typing JSON, open the piano-roll editor — a lighter, polyphonic canvas with voice-aware MIDI export. For rhythm, open Cycla: a companion builder for .cyc files, treating meter as a recursive subdivision grammar — a tuning system for time, parallel to how Scala describes tuning systems for pitch.

Score Source

Compiles one monophonic voice to a Standard MIDI File. Each note is placed on the nearest 12-TET key and retuned with a single per-channel pitch bend, so any microtonal frequency transfers exactly. The file writes an RPN 0,0 message that tells the receiving synth what bend range to expect — set it to match your instrument.
